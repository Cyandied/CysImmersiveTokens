How do I create tokens?

The layers

Its simple, relativly, it is all about the layers, and you'll need plenty. To create a finished token like the one to the left, I needed 6 layers.
  • lines and static colors
  • alpha top
  • alpha middle
  • alpha bottom
  • glow
  • background
To see the separation, click the token, and it will split up like a burger. I have made a few decesions here, for starters, i have decided to collect any static coloring (shadows, highlight, other) in one layer with the line-art. This is a way to control the amount of layers you are working with. I also decided to separate my two background elements, glow and background. This is so I can easily reuse the same background for several different tokens.
To learn more about how to format any of the layers, simply click it, and I'll tell you a bit more in deltail how you should go about making them.

Naming conventions

If you plan to use this service for a large amount of tokens, save yourself some pain and stick to a rigid naming scheme, I reccomend:
  • example_lines_static.extention
  • example_alpha_A.extention
  • example_alpha_B.extention
  • example_alpha_C.extention
  • example_glow.extention
  • basic_background.extention
It is clear what is what, the order of the alpahs are nicely seprated via the alphabet (top A → Z bottom), and my basic background, which I am planning to reuse many times, is easily dicrnable so you dont accidentally purge it.
As an extra help, make absolutely sure you name the background with the _background or _bg added on at the very end of its filename. Then it will remain in the editor after pressing "purge top".
When uploading images, ensure that they are the same size, all of them! Else the whole editor breaks...
